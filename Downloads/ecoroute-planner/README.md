# EcoRoute Planner

Your Quickest Path to Green Transit. Compare travel modes, estimate time, and visualize carbon footprint savings to make eco-friendly travel decisions using Gemini AI.

## Features

- **Multi-Mode Analysis:** Compare Car, EV, Bus, Train, Bike, and Walking.
- **AI-Powered Routes:** Uses Gemini 2.5 Flash to calculate routes, emissions, and costs.
- **Real-time Data:** Includes real-time pollution gauges (via Open-Meteo) and simulated traffic analysis.
- **Interactive Map:** Visualize routes and waypoints on a Leaflet map.
- **Localization:** Support for English and Local languages (e.g., Arabic, French).
